Welcome to the C++ Learning Section!

In this section, you'll find tutorials, examples, and explanations to help you learn C++ programming language.

Basic C++ Syntax


#include <iostream>

int main() {
    // Print Hello World to the console
    std::cout << "Hello World!" << std::endl;
    return 0;
}

In the code above:

  • #include <iostream>: This line includes the standard input/output stream library, which provides functionality for input and output operations.
  • int main(): This is the main function of the program. It serves as the entry point for execution.
  • std::cout << "Hello World!" << std::endl;: This line prints "Hello World!" to the console using the std::cout stream. The << operator is used for output, and std::endl is used to insert a newline character.
  • return 0;: This line indicates that the program execution was successful and returns a status code of 0 to the operating system.

C++ Variables

In C++, variables are used to store data. They have a specific data type that determines what kind of data they can hold. Here's an example:


#include <iostream>

int main() {
    // Declare and initialize a variable
    int num = 10;
    
    // Print the value of the variable
    std::cout << "The value of num is: " << num << std::endl;
    
    return 0;
}

In the code above:

  • #include <iostream>: This line includes the standard input/output stream library, which provides functionality for input and output operations.
  • int main(): This is the main function of the program. It serves as the entry point for execution.
  • int num = 10;: This line declares a variable named num of type int and initializes it with the value 10.
  • std::cout << "The value of num is: " << num << std::endl;: This line prints the value of the variable num to the console using the std::cout stream.
  • return 0;: This line indicates that the program execution was successful and returns a status code of 0 to the operating system.

Data Types in C++

In C++, data types specify the type of data that variables can hold. C++ provides several built-in data types to handle different types of data. These data types can be broadly categorized into the following:

  • Primitive Data Types: Basic data types provided by the language.
  • Derived Data Types: Derived from primitive data types or other derived data types.
  • User-defined Data Types: Defined by the user.

Commonly used primitive data types in C++ include:

  • int: Integer type for whole numbers.
  • float: Floating-point type for decimal numbers.
  • double: Double-precision floating-point type.
  • char: Character type for single characters.
  • bool: Boolean type representing true or false values.

Here's an example demonstrating the use of primitive data types in C++:


#include <iostream>
int main() {
    int age = 25;
    float height = 5.8;
    char grade = 'A';
    bool isStudent = true;

    std::cout << "Age: " << age << std::endl;
    std::cout << "Height: " << height << std::endl;
    std::cout << "Grade: " << grade << std::endl;
    std::cout << "Is Student? " << isStudent << std::endl;

    return 0;
}

In this example, we declare variables of different data types (int, float, char, bool) and print their values to the console.

Operators in C++

In C++, operators are symbols that perform operations on operands. C++ supports a wide range of operators, which can be categorized into the following types:

  • Arithmetic Operators: Perform arithmetic operations like addition, subtraction, multiplication, division, etc.
  • Relational Operators: Compare the values of operands.
  • Logical Operators: Perform logical operations on Boolean operands.
  • Assignment Operators: Assign values to variables.
  • Increment and Decrement Operators: Increment or decrement the value of a variable.
  • Conditional Operator: Also known as the ternary operator, it is used for conditional expressions.
  • Bitwise Operators: Perform bitwise operations on binary representations of operands.

Here's an example demonstrating the usage of various operators in C++:


#include <iostream>

int main() {
    int a = 10, b = 5;
    int sum = a + b; // Addition
    int difference = a - b; // Subtraction
    int product = a * b; // Multiplication
    int quotient = a / b; // Division
    int remainder = a % b; // Modulus

    bool isEqual = (a == b); // Equal to
    bool isGreater = (a > b); // Greater than
    bool isLess = (a < b); // Less than

    bool logicalAnd = (a && b); // Logical AND
    bool logicalOr = (a || b); // Logical OR
    bool logicalNot = !a; // Logical NOT

    int x = 5;
    x += 2; // Compound assignment (x = x + 2)
    x++; // Increment
    x--; // Decrement

    int result = (a > b) ? a : b; // Conditional operator

    int bitwiseAnd = a & b; // Bitwise AND
    int bitwiseOr = a | b; // Bitwise OR
    int bitwiseXor = a ^ b; // Bitwise XOR
    int bitwiseComplement = ~a; // Bitwise complement
    int leftShift = a << 1; // Left shift
    int rightShift = a >> 1; // Right shift

    std::cout << "Sum: " << sum << std::endl;
    std::cout << "Difference: " << difference << std::endl;
    std::cout << "Product: " << product << std::endl;
    std::cout << "Quotient: " << quotient << std::endl;
    std::cout << "Remainder: " << remainder << std::endl;

    // Print other operator results...

    return 0;
}

This example demonstrates the usage of various arithmetic, relational, logical, assignment, increment, decrement, conditional, and bitwise operators in C++.

If-Else Statement in C++

In C++, the if statement is used for conditional execution of code based on a condition. If the condition evaluates to true, the code inside the if block is executed; otherwise, it is skipped. Optionally, you can use the else statement to specify an alternative code block to execute if the condition evaluates to false.

The syntax of the if statement is as follows:


if (condition) {
    // Code to be executed if condition is true
} else {
    // Code to be executed if condition is false
}

Here's an example demonstrating the usage of the if-else statement in C++:


#include <iostream>

int main() {
    int age = 20;

    if (age >= 18) {
        std::cout << "You are eligible to vote." << std::endl;
    } else {
        std::cout << "You are not eligible to vote." << std::endl;
    }

    return 0;
}

In this example, if the value of age is greater than or equal to 18, the message "You are eligible to vote." is printed; otherwise, the message "You are not eligible to vote." is printed.

Switch Statement in C++

In C++, the switch statement is used for multi-way branching. It provides a convenient way to execute different blocks of code based on the value of an expression.

The syntax of the switch statement is as follows:


switch (expression) {
    case constant1:
        // Code to be executed if expression equals constant1
        break;
    case constant2:
        // Code to be executed if expression equals constant2
        break;
    // Add more cases as needed...
    default:
        // Code to be executed if expression does not match any case
}

Here's an example demonstrating the usage of the switch statement in C++:


#include <iostream>

int main() {
    int day = 3;
    std::string dayName;

    switch (day) {
        case 1:
            dayName = "Monday";
            break;
        case 2:
            dayName = "Tuesday";
            break;
        case 3:
            dayName = "Wednesday";
            break;
        case 4:
            dayName = "Thursday";
            break;
        case 5:
            dayName = "Friday";
            break;
        case 6:
            dayName = "Saturday";
            break;
        case 7:
            dayName = "Sunday";
            break;
        default:
            dayName = "Invalid day";
    }

    std::cout << "Day: " << dayName << std::endl;

    return 0;
}

In this example, based on the value of the day variable, different blocks of code are executed to assign the corresponding day name to the dayName variable.

Loop in C++

In C++, loops are used to execute a block of code repeatedly until a specified condition is met. C++ provides several types of loops:

  • for loop: Executes a block of code a specified number of times.
  • while loop: Executes a block of code repeatedly as long as a given condition is true.
  • do-while loop: Executes a block of code once and then repeatedly executes the block as long as a given condition is true.

Here's an example demonstrating the use of the for loop in C++:


#include <iostream>

int main() {
    // Example: Print numbers from 1 to 5
    for (int i = 1; i <= 5; i++) {
        std::cout << i << std::endl;
    }

    return 0;
}

In this example, we use a for loop to print numbers from 1 to 5. The loop starts with initializing i to 1, checks the condition i <= 5, executes the block of code, and increments i after each iteration.

Function in C++

In C++, a function is a block of code that performs a specific task. Functions provide modularity and reusability to a program by allowing you to encapsulate code into manageable units.

The syntax of a function declaration in C++ is as follows:


return_type function_name(parameter_list) {
    // Function body
}

Here's an example demonstrating the declaration and usage of a function in C++:


#include <iostream>

// Function declaration
int add(int a, int b) {
    return a + b;
}

int main() {
    int num1 = 5, num2 = 3;
    
    // Function call
    int sum = add(num1, num2);
    
    std::cout << "Sum: " << sum << std::endl;

    return 0;
}

In this example, we define a function add that takes two integer parameters a and b and returns their sum. We then call this function from the main function to compute the sum of two numbers.

Arrays in C++

In C++, an array is a collection of elements of the same data type stored in contiguous memory locations. Arrays provide a convenient way to store and access multiple values of the same type.

The syntax to declare an array in C++ is as follows:


data_type array_name[array_size];

Here's an example demonstrating the declaration and usage of arrays in C++:


#include <iostream>

int main() {
    // Declaration and initialization of an array
    int numbers[5] = {10, 20, 30, 40, 50};

    // Accessing elements of the array
    std::cout << "First element: " << numbers[0] << std::endl;
    std::cout << "Third element: " << numbers[2] << std::endl;

    // Modifying elements of the array
    numbers[1] = 25;

    // Printing all elements of the array using a loop
    std::cout << "Elements of the array:" << std::endl;
    for (int i = 0; i < 5; i++) {
        std::cout << numbers[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}

In this example:

  • We declare and initialize an array numbers with five elements.
  • We access elements of the array using index notation numbers[index].
  • We modify an element of the array by assigning a new value to it.
  • We use a loop to print all elements of the array.

Arrays in C++ are zero-indexed, meaning the first element has an index of 0, the second element has an index of 1, and so on.

In addition to single-dimensional arrays, C++ also supports multi-dimensional arrays and dynamic arrays using pointers.

Pointers in C++

In C++, a pointer is a variable that stores the memory address of another variable. Pointers provide a way to directly access and manipulate memory locations, which can be useful for dynamic memory allocation, passing parameters by reference, and working with arrays and data structures.

The syntax to declare a pointer variable in C++ is as follows:


data_type *pointer_name;

Here's an example demonstrating the declaration and usage of pointers in C++:


#include <iostream>

int main() {
    int num = 10;
    int *ptr;

    // Assigning the address of num to the pointer
    ptr = #

    // Accessing the value of num using the pointer
    std::cout << "Value of num: " << *ptr << std::endl;

    // Modifying the value of num using the pointer
    *ptr = 20;

    // Printing the new value of num
    std::cout << "New value of num: " << num << std::endl;

    return 0;
}

In this example:

  • We declare a variable num and a pointer ptr of type int.
  • We assign the address of num to the pointer using the & (address-of) operator.
  • We access the value of num using the pointer dereferencing operator *.
  • We modify the value of num indirectly through the pointer.

Pointers can also be used to dynamically allocate memory using the new operator and deallocate memory using the delete operator.

Object and Class in C++

In C++, a class is a blueprint for creating objects. It defines the properties (data members) and behaviors (member functions) of objects of that type. An object is an instance of a class, which represents a specific entity in a program.

The syntax to define a class in C++ is as follows:


class ClassName {
    // Data members (properties)
    data_type member1;
    data_type member2;
    // ...

public:
    // Member functions (behaviors)
    return_type memberFunction1(parameter_list) {
        // Function body
    }
    return_type memberFunction2(parameter_list) {
        // Function body
    }
    // ...
};

Here's an example demonstrating the declaration and usage of a class and objects in C++:


#include <iostream>

// Define a class
class Rectangle {
private:
    // Data members (private by default)
    int length;
    int width;

public:
    // Constructor
    Rectangle(int l, int w) {
        length = l;
        width = w;
    }

    // Member function to calculate area
    int area() {
        return length * width;
    }
};

int main() {
    // Create objects of the class
    Rectangle rect1(5, 3);
    Rectangle rect2(4, 6);

    // Call member function to calculate area
    std::cout << "Area of rect1: " << rect1.area() << std::endl;
    std::cout << "Area of rect2: " << rect2.area() << std::endl;

    return 0;
}

In this example:

  • We define a class Rectangle with data members length and width and a member function area() to calculate the area of a rectangle.
  • We create two objects rect1 and rect2 of the class Rectangle.
  • We call the member function area() on each object to calculate the area of the rectangles.

Classes in C++ support encapsulation, inheritance, and polymorphism, providing powerful tools for building complex systems.

    Inheritance in C++

    In C++, inheritance is a mechanism by which a class can inherit properties and behavior (member variables and member functions) from another class. The class that is inherited from is called the base class or parent class, and the class that inherits is called the derived class or child class.

    The syntax to define a derived class in C++ using inheritance is as follows:

    
class DerivedClassName : access_specifier BaseClassName {
    // Additional member variables and functions
};

    Here's an example demonstrating inheritance in C++:

    
#include <iostream>

// Base class
class Shape {
protected:
    int width;
    int height;

public:
    Shape(int w, int h) {
        width = w;
        height = h;
    }

    void setWidth(int w) {
        width = w;
    }

    void setHeight(int h) {
        height = h;
    }
};

// Derived class
class Rectangle : public Shape {
public:
    Rectangle(int w, int h) : Shape(w, h) {}

    int area() {
        return width * height;
    }
};

int main() {
    Rectangle rect(5, 3);

    std::cout << "Area of rectangle: " << rect.area() << std::endl;

    return 0;
}

    In this example:

    • We define a base class Shape with member variables width and height and member functions to set their values.
    • We define a derived class Rectangle that inherits from the base class Shape.
    • The derived class Rectangle has access to the member variables and functions of the base class Shape.
    • We create an object rect of the derived class Rectangle and call its member function area() to calculate the area.

    Inheritance in C++ supports single inheritance, multiple inheritance, hierarchical inheritance, and multilevel inheritance, providing flexibility in designing class hierarchies.
