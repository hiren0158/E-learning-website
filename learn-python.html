Basic Python Syntax


# Python code to print "Hello, World!" to the console
print("Hello, World!")

Python is a high-level programming language known for its simplicity and readability. Let's break down the code above:

  • #: The hash symbol indicates a comment in Python. Comments are ignored by the interpreter and are used to add notes or explanations to the code.
  • print(): This is a built-in Python function used to print messages to the console. In this case, it prints the string "Hello, World!".
  • "Hello, World!": This is a string literal, enclosed in double quotes, representing the message that will be printed.

Python uses indentation to define blocks of code. This indentation is typically four spaces or one tab. It's important to maintain consistent indentation to ensure proper code execution.

Now, let's explore some more basic Python syntax and features:

Variables in Python

In Python, variables are used to store data values. Unlike some other programming languages, Python does not require declaring variables with a specific data type. The type of a variable is dynamically determined based on the value assigned to it.

Here are some important points about variables in Python:

  • Naming Rules: Variable names can contain letters (a-z, A-Z), digits (0-9), and underscores (_). They cannot start with a digit and cannot be a Python keyword.
  • Case-Sensitivity: Variable names are case-sensitive. For example, myVar, MyVar, and MYVAR are treated as different variables.
  • Assignment: Variables are created when they are first assigned a value. You can assign a value to a variable using the assignment operator (=).
  • Data Types: Python supports various data types such as integers, floats, strings, lists, tuples, dictionaries, etc. Variables can hold values of any data type.
  • Dynamic Typing: Python is dynamically typed, meaning you don't have to declare the type of a variable explicitly. The interpreter automatically determines the type based on the value assigned to it.

Here's an example demonstrating the usage of variables in Python:


# Assigning values to variables
name = "Alice"
age = 30
height = 5.8
is_student = False

# Displaying variable values
print("Name:", name)
print("Age:", age)
print("Height:", height)
print("Is student:", is_student)

In this example:

  • We assign values of different data types (string, integer, float, boolean) to variables.
  • We print the values of these variables using the print() function.

Python's flexibility with variables makes it easy to work with different types of data and simplifies the development process.

Input Function in Python

In Python, the input() function is used to prompt the user to enter input from the keyboard. The input entered by the user is treated as a string by default.

The syntax of the input() function is as follows:


variable_name = input(prompt_message)

Here's an example demonstrating the usage of the input() function in Python:


# Prompt the user to enter their name
name = input("Enter your name: ")

# Prompt the user to enter their age
age = input("Enter your age: ")

# Display the user's information
print("Hello, " + name + "! You are " + age + " years old.")

In this example:

  • We use the input() function to prompt the user to enter their name and age.
  • The input entered by the user is stored in variables name and age.
  • We then display a personalized message using the user's name and age.

It's important to note that the input() function always returns a string. If you need to convert the input to a different data type, you can use type conversion functions such as int(), float(), etc.

Data Types in Python

Python supports various built-in data types to store different kinds of data. Some of the commonly used data types in Python include:

  • Integer (int): Represents whole numbers, positive or negative, without any decimal point.
  • Float (float): Represents real numbers with a decimal point.
  • String (str): Represents a sequence of characters enclosed within single quotes (' ') or double quotes (" ").
  • List: Represents an ordered collection of items. Lists are mutable, meaning their elements can be changed after creation.
  • Tuple: Similar to lists, tuples represent an ordered collection of items, but they are immutable, meaning their elements cannot be changed after creation.
  • Dictionary (dict): Represents a collection of key-value pairs. Keys are unique and immutable, and values can be of any data type.
  • Boolean (bool): Represents the truth values True or False.
  • NoneType (None): Represents the absence of a value or a null value.

Here's an example demonstrating the usage of different data types in Python:


# Integer
num1 = 10

# Float
num2 = 3.14

# String
name = "Alice"

# List
numbers = [1, 2, 3, 4, 5]

# Tuple
coordinates = (10, 20)

# Dictionary
person = {"name": "Bob", "age": 30, "is_student": True}

# Boolean
is_adult = True

# NoneType
empty_value = None

In this example:

  • We declare variables of different data types including integers, floats, strings, lists, tuples, dictionaries, booleans, and None.
  • We assign values to these variables using appropriate syntax for each data type.

Understanding the different data types in Python is crucial for effectively manipulating and processing data in Python programs.

round() Function in Python

The round() function in Python is used to round a given number to a specified number of decimal places or to the nearest integer.

The syntax of the round() function is as follows:


round(number, ndigits=None)

Where:

  • number is the number to be rounded.
  • ndigits (optional) is the number of decimal places to round to. If omitted, ndigits defaults to 0, rounding to the nearest integer.

Here are some examples demonstrating the usage of the round() function:


# Round to the nearest integer
print(round(3.14))  # Output: 3
print(round(3.78))  # Output: 4

# Round to a specified number of decimal places
print(round(3.1415926535, 2))  # Output: 3.14
print(round(3.1415926535, 4))  # Output: 3.1416

In the first example, the round() function is used without specifying the number of decimal places, so it rounds the number to the nearest integer.

In the second example, the round() function is used with ndigits=2, so it rounds the number to 2 decimal places.

The round() function follows the standard rounding rules: if the decimal part is exactly halfway between two integers, it rounds to the nearest even integer.

if-else Statement in Python

The if-else statement in Python is used to execute a block of code conditionally based on whether a given condition is true or false.

The syntax of the if-else statement is as follows:


if condition:
    # Code block to execute if the condition is true
else:
    # Code block to execute if the condition is false

Here's an example demonstrating the usage of the if-else statement in Python:


# Example: Check if a number is positive or negative
num = -10

if num > 0:
    print("The number is positive.")
else:
    print("The number is negative.")

In this example:

  • We use the if-else statement to check if the variable num is greater than zero.
  • If the condition num > 0 is true, the message "The number is positive." is printed.
  • Otherwise, if the condition is false, the message "The number is negative." is printed.

The if-else statement can be followed by one or more elif (else if) clauses to check multiple conditions sequentially. It allows for more complex conditional logic in Python programs.
